 Step 1: Go to OpenAI's Platform
Visit: https://platform.openai.com/

🔹 Step 2: Sign Up / Log In
Click "Sign Up" if you don’t have an account.
Or click "Log In" if you already have one.
You can sign in using:
Google account
Microsoft account
Or email/password.

🔹 Step 3: Verify Your Email (if new)
If you signed up just now, OpenAI will send a verification link to your email.
Open your inbox and click the link to verify.

🔹 Step 4: Go to API Key Section
After login, go to:
👉 https://platform.openai.com/account/api-keys

🔹 Step 5: Create New API Key
Click the "Create new secret key" button.

A new API key will appear (something like sk-xxxxxxxxxxxx).
⚠️ Copy and save it immediately. You won’t see it again.
